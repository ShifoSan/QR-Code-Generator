# QR Code Generator

A modern, feature-rich QR code generator web app.

## Features (Planned)
- Generate QR codes for text, URLs, WiFi, and other data
- High-res export (PNG/SVG)
- Modern UI, animations, and dark mode
- Custom color and background options
- Dynamic preview
- Download and share options
- History of generated codes
- Mobile-friendly design

## Usage
Just enter your text, click 'Generate QR', and your QR code will appear with instant preview.
